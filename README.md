# Django tutorial

## Install Django
```
cd C:\....\Python\Python35-32\Scripts
pip install Django
```

## Create New Django project
```
django-admin.py startproject Brain_Garden
```

## Runserver
```
manage.py runserver (port)
```

## Add Post model
```
manage.py startapp posts
```

## Make migrations
```
manage.py makemigrations
```

## Update migrations
```
manage.py migrate
```
